jQuery Quiz 01

             1. In jQuery terminology,what is the difference  between the “jQuery function” and 
                the “jQuery object”?

Answer: JQuery function is a function that returns a jQuery Object.
But jQuery object most of the time represents elements of the DOM. 

            2. Write jQuery code to find all h1 elements that are children of a div element and make their background
                color red.

Answer: $("div p").css("background-color", "red") 

            3. Use a jQuery method to insert the text "YES!" at the end of the 'p' element.

Answer: $("p").append(" YES! ")
